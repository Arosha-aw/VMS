# 🎬 Video Management System (VMS)

A simple yet powerful Excel-based system to manage and track lecture recordings efficiently.


## 📌 Overview

**Video Management System (VMS)** is an Excel application designed to help you organize, track, and monitor lecture recordings in a structured way.

It combines a user-friendly interface, automated data storage, and a visual dashboard to help you stay consistent and focused.



## ✨ Features

* 📝 Easy Data Entry Form
* 📊 Dashboard for progress tracking
* 📂 Automatic data storage in a table
* ⏱️ Track total hours vs. watched hours
* 🔗 Identify missing video links
* 📅 Recent video tracking with date
* 🔍 Filter by subject for focused learning

## 🛠️ Built With

* Microsoft Excel (Macros/VBA)



## 🚀 Usage

1. Enter lecture details using the data entry form
2. Data is automatically stored in the table
3. Use filters to organize by subject
4. Track your progress using the dashboard



## ⚙️ Installation

1. Download the Excel file
2. Open it in Microsoft Excel
3. Enable **"Macro Content"** when prompted ⚠️
4. (Optional) Use **Dark Mode** for better experience
